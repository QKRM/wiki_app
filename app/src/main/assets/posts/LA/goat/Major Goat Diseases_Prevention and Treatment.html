1. ການສັງເກດ, ການວິນິດໄສ, ແລະການປ້ອງກັນພະຍາດ

ແບ້ມັກຈະ ເຊື່ອງອາການຂອງພະຍາດ , ເຮັດໃຫ້ມັນຍາກທີ່ຈະກວດພົບອາການເບື້ອງຕົ້ນ. ໃນຫຼາຍໆກໍລະນີ, ການເຈັບປ່ວຍບໍ່ໄດ້ສັງເກດເຫັນຈົນກ່ວາສະພາບຈະກາຍເປັນຮ້າຍແຮງ, ໃນເວລານັ້ນ ການປິ່ນປົວອາດຈະມີປະສິດທິພາບຫນ້ອຍ .

ດັ່ງນັ້ນ, ການກວດຫາເບື້ອງຕົ້ນ ໂດຍຜ່ານການສັງເກດເປັນປົກກະຕິແມ່ນເປັນສິ່ງຈໍາເປັນ. ຢ່າງໃດກໍ່ຕາມ, ສິ່ງສໍາຄັນຍິ່ງກວ່ານັ້ນແມ່ນ ການປ້ອງກັນພະຍາດ ໂດຍຜ່ານການໃຫ້ອາຫານທີ່ເຫມາະສົມ, ການຄຸ້ມຄອງ, ແລະການອະນາໄມ.

□ ຄໍາແນະນໍາສໍາລັບການຕິດຕາມສຸຂະພາບໃນແບ້

ສຸຂະພາບຂອງແບ້ສາມາດປະເມີນໄດ້ໂດຍຜ່ານ ການສັງເກດປະຈໍາວັນຢ່າງລະມັດລະວັງ ຂອງຕົວຊີ້ວັດທາງດ້ານຮ່າງກາຍແລະພຶດຕິກໍາຕ່າງໆ. ອາການຕໍ່ໄປນີ້ຄວນໄດ້ຮັບການກວດກາເປັນປົກກະຕິໂດຍຜູ້ດູແລ:

  • ການ​ເຄື່ອນ​ໄຫວ (ຂາ​ຂາ​ຫຼື​ແຂງ​)
  • ຄວາມຢາກອາຫານ (ອາຫານຫຼຸດລົງຫຼືການໄດ້ຮັບນ້ໍາ)
  • ສະພາບຂອງເປືອກຫຸ້ມນອກ (ເງົາ, ໂຄງສ້າງ, ຫຼືການສູນເສຍ patchy)
  • ນ້ ຳ ໜັກ ຂອງຮ່າງກາຍປ່ຽນແປງ
  • ສະພາບຂອງອາຈົມ ແລະປັດສະວະ (ຄວາມສອດຄ່ອງ, ຄວາມຖີ່, ມີເລືອດ ຫຼືນໍ້າເມືອກ)
  • ຕາ (ຄວາມແຈ່ມແຈ້ງ, ປະກົດການໄຫຼ, ຟັງ)
  • ອຸນຫະພູມຮ່າງກາຍ
  • ຮູຂຸມຂົນ (ການເຕີບໃຫຍ່, ກິ່ນເໝັນ, ຂີ້ຄ້ານ)
  • ບາດແຜ ຫຼື ບວມ
  • ນ້ຳອອກທາງດັງ ຫຼື ຫາຍໃຈຝືດ
  • ນ້ຳຕາໄຫຼອອກຜິດປົກກະຕິ

⚠️ ການສັງເກດຢ່າງເປັນປົກກະຕິ ແລະ ເປັນລະບົບ ຊ່ວຍໃຫ້ສາມາດ ກວດພົບ ແລະ ປິ່ນປົວໄດ້ໄວ , ເຊິ່ງເປັນສິ່ງສໍາຄັນໃນການຮັກສາສຸຂະພາບຂອງຝູງສັດ ແລະ ຫຼຸດຜ່ອນອັດຕາການຕາຍ.

□ ລາຍການກວດກາສຸຂະພາບແບ້

2. ການປ້ອງກັນພະຍາດ ແລະ ພະຍາດແບ້ທົ່ວໄປ

□ ການປ້ອງກັນພະຍາດ

  • ໃຫ້ແບ້ດ້ວຍ ອາຫານທີ່ອຸດົມດ້ວຍສານອາຫານ ແລະ ປ່ອຍ ທົ່ງຫຍ້າລ້ຽງສັດ ໃຫ້ຫຼາຍເທົ່າທີ່ຈະເຮັດໄດ້ເພື່ອໃຫ້ພວກມັນກິນອາຫານທີ່ຕ້ອງການ.
  • ເກັບຮັກສາ ແບ້ໃຫ້ແຫ້ງ ເພື່ອປ້ອງກັນຄວາມຊຸ່ມຊື່ນ. ໃນລະດູຫນາວ, ສະກັດຮ່າງເຢັນແລະຮັກສາຄວາມອົບອຸ່ນໃນຂະນະທີ່ຮັບປະກັນການລະບາຍອາກາດ. ໃນລະດູຮ້ອນ, ຮັບປະກັນ ການໄຫຼວຽນຂອງອາກາດທີ່ດີ ແລະຄວາມເຢັນ.
  • ປົກປ້ອງແບ້ຈາກ ຍຸງກັດ , ໂດຍສະເພາະໃນລະດູຮ້ອນ.
  • ໃຫ້ແນ່ໃຈວ່າແບ້ໄດ້ຮັບ ການອອກກໍາລັງກາຍຢ່າງພຽງພໍ .
  • ຄຸ້ມຄອງ ແມ່ທ້ອງໃນລະດູໃບໄມ້ປົ່ງ ແລະ ລະດູໃບໄມ້ປົ່ງ , ແລະ ຂ້າເຊື້ອໃນສວນ ແລະ ເດີ່ນຫຍ້າ ເປັນປະຈຳ ເພື່ອປ້ອງກັນການຕິດເຊື້ອແມ່ກາຝາກ.

□ ພະຍາດແບ້ທົ່ວໄປ ແລະການປິ່ນປົວ

○ ທ້ອງອືດ (Acute Bloat (Ruminal Tympany))

  • ສາເຫດ : ອາຫານທີ່ເສື່ອມເສຍ ຫຼື ໝັກດອງ, ອາຫານທີ່ປຽກຝົນ, ພືດຕະກຸນຫຼາຍ, ການປ່ຽນອາຫານກະທັນຫັນ (ມັກໃນລະດູຮ້ອນ).
  • ອາການ : ທ້ອງຊ້າຍບວມ, ບໍ່ມີຄາບ, ຫາຍໃຈຍາກ.
  • ການປິ່ນປົວ :
  • ອ່ອນໆ : ເອົາໄມ້ ຫຼື ຊິ້ນໄມ້ໃສ່ປາກເພື່ອສົ່ງເສີມການປ່ອຍອາຍແກັສ; ໃຫ້ອາຊິດ hydrochloric 3 mL + 3-9 mL ethyl alcohol ເຈືອຈາງໃນນ້ໍາ 5 ສ່ວນ.
  • ຮຸນແຮງ : ສຽບທໍ່ ຫຼືທໍ່ເຂົ້າໄປໃນທ້ອງຊ້າຍເພື່ອປ່ອຍອາຍແກັສ, ຈາກນັ້ນໃຊ້ຢາຕ້ານການໝັກ (ເຊັ່ນ: ແກສຕຼິດ, ແກສມິນ, ແກສມິນ).
  • ການປ້ອງກັນ : ຫຼີກລ່ຽງການໃຫ້ອາຫານສັດທີ່ເສື່ອມເສຍ, ໝັກດອງ ຫຼື ພືດທີ່ອຸດົມສົມບູນໄປດ້ວຍພືດຕະກຸນຖົ່ວໃນຄັ້ງດຽວ.

○ Ruminal Stasis (ຜົນກະທົບ)

  • ສາເຫດ : ການໃຫ້ອາຫານຫຼາຍເກີນໄປ, ການປ່ຽນແປງອາຫານຢ່າງກະທັນຫັນ, ຮ່າງກາຍອ່ອນເພຍ (ມັກພົບໃນລະດູຫນາວ).
  • ອາການ : ຂາດຄວາມຢາກອາຫານ, ບໍ່ມີອາຫານ, ປວດຫລັງ, ຫາຍໃຈໄວ.
  • ການປິ່ນປົວ : ປະຕິບັດຢາກະຕຸ້ນ rumen ເຊັ່ນ Tumix, Tafumin, ຫຼື Torumin; ນວດທ້ອງຊ້າຍເພື່ອຟື້ນຟູການເຄື່ອນໄຫວຂອງ rumen; ປິ່ນປົວທ້ອງອືດຖ້າພ້ອມກັນ.
  • ການປ້ອງກັນ : ແນະນຳການປ່ຽນແປງອາຫານເທື່ອລະກ້າວ; ຫຼີກເວັ້ນການໃຫ້ອາຫານເກີນ.

○ ຫຼອດລົມອັກເສບ / ຫຼອດລົມອັກເສບ

  • ສາເຫດ : ການປ່ຽນແປງດິນຟ້າອາກາດຢ່າງກະທັນຫັນ, ການຫາຍໃຈເອົາທາດອາຍພິດທີ່ລະຄາຍເຄືອງ (ໂດຍປົກກະຕິໃນລະດູຫນາວ).
  • ອາການ : ໄອ, ດັງ, ຫາຍໃຈຍາກ.
  • ການປິ່ນປົວ : ໃຊ້ຢາຕ້ານເຊື້ອ (ຕົວຢ່າງ: codeine phosphate, morphine hydrochloride), ຢາຕ້ານໄຂ້ (ເຊັ່ນ: ແອດສະໄພລິນ, APC), ແລະຢາຕ້ານເຊື້ອ (ເຊັ່ນ: ເປນີຊີລິນ, ເທຣາໄມຊິນ, ເຈນທາມິຊິນ, ລິນໂຄໄມຊິນ).
  • ການປ້ອງກັນ : ຮັກສາຄວາມຊຸ່ມຊື່ນທີ່ເຫມາະສົມ, ການລະບາຍອາກາດ, ແລະອຸນຫະພູມທີ່ຫມັ້ນຄົງໃນທີ່ຢູ່ອາໄສ.

○ ປອດອັກເສບ

  • ສາເຫດ : ສາເຫດທີ່ສອງຈາກການຕິດເຊື້ອໃນ bronchial catarrh ຫຼື Pasteurella .
  • ອາການ : ມີໄຂ້ສູງ, ໄອ, ຫາຍໃຈຍາກ.
  • ການ​ປິ່ນ​ປົວ : ຢາ​ຕ້ານ​ເຊື້ອ intramuscular ເຊັ່ນ gentamicin, lincomycin, kanamycin, tylosin.
  • ການປ້ອງກັນ : ກໍາຈັດສາເຫດທີ່ປະກອບສ່ວນ (ອາກາດເຢັນ, ລະບາຍອາກາດບໍ່ດີ).

○ ອໍາມະພາດກະດູກສັນຫຼັງ (ການຕິດເຊື້ອ Parelaphostrongylus)

  • ສາເຫດ : ການຕິດເຊື້ອ Microfilariae ຜ່ານຍຸງ.
  • ລະດູການ : ເດືອນສິງຫາ – ກັນຍາ.
  • ອາການ : ຫົວອຽງ, ອຳມະພາດຂອງແຂນຂາ ແລະ ຂາຫຼັງ.
  • ການປິ່ນປົວ : ສັກຢາ Spaton Intramuscular ສໍາລັບ 4-5 ມື້.
  • ການປ້ອງກັນ : ປ້ອງກັນຍຸງກັດ.

○ ພຍາດປອດ (Dictyocaulus infection)

  • ສາເຫດ : ການຕິດເຊື້ອແມ່ທ້ອງຜ່ານຫອຍ → ທົ່ງຫຍ້າທີ່ປົນເປື້ອນ.
  • ອາການ : ໄອ, ຫາຍໃຈຍາກ, ຫຼອດລົມອັກເສບຊໍາເຮື້ອ; ກໍລະນີຊໍາເຮື້ອສະແດງໃຫ້ເຫັນການຂາດສານອາຫານແລະຖອກທ້ອງ.
  • ການປິ່ນປົວ : Deworm ກັບຢາເຊັ່ນ Ripacol.
  • ການປ້ອງກັນ : ຫຼີກລ້ຽງທົ່ງຫຍ້າລ້ຽງສັດທີ່ມີແມງໄມ້.

○ ແມ່ທ້ອງ (Haemonchosis)

  • ລະດູການ : ພົບທົ່ວໄປໃນທ້າຍລະດູໃບໄມ້ປົ່ງ.
  • ອາການ : ນ້ຳໜັກຫຼຸດລົງ, ພະຍາດເລືອດຈາງ.
  • ການປິ່ນປົວ : Deworm ດ້ວຍຢາເຊັ່ນ: Rital ຫຼື Tetrasol-20.
  • ການ​ປ້ອງ​ກັນ : ຫຼີກ​ລ້ຽງ​ທົ່ງ​ຫຍ້າ​ທີ່​ປົນ​ເປື້ອນ​ຫຼື overgrazed​; ໃຊ້ແຫຼ່ງນໍ້າສະອາດ.

○ ພະຍາດຕັບອັກເສບ (Fascioliasis)

  • ສາເຫດ : ພະຍາດຕັບອັກເສບ ( Fasciola , Dicrocoelium , Paramphistomum ) ໃນທໍ່ນໍ້າບີ.
  • ເສັ້ນທາງການຕິດເຊື້ອ : ແບ້ → ຫອຍ → ຫຍ້າທີ່ປົນເປື້ອນ → ແບ້.
  • ອາການ : ໄຂ້ອ່ອນໆ, ເບື່ອອາຫານ, ຂາດສານອາຫານ, ເລືອດຈາງ.
  • ການປິ່ນປົວ : ປະຕິບັດການປິ່ນປົວພະຍາດຕັບອັກເສບເຊັ່ນ Bichinol, Nilzan ທາງປາກ; Bilibon intramuscularly.
  • ການປ້ອງກັນ : ກໍາຈັດສັດຕູພືດກາງ (ຫອຍ); ຫຼີກລ້ຽງທົ່ງຫຍ້າລ້ຽງສັດ.

○ ຜິວໜັງອັກເສບ (Infectious Conjunctivitis)

  • ສາເຫດ : ການຕິດເຊື້ອແບັກທີເລຍ Moraxella , ແຜ່ລາມໂດຍແມງວັນ ແລະແມງໄມ້.
  • ອາການ : ແສບຕາ, ຕົກຂາວ, ຕາແດງ; opacity corneal ແລະ ulcers ອາດຈະເຮັດໃຫ້ຕາບອດ.
  • ການປິ່ນປົວ :
  • ແຍກແບ້ທີ່ຖືກກະທົບ.
  • ລ້າງຕາເລື້ອຍໆດ້ວຍການແກ້ໄຂອາຊິດ boric 3%.
  • ທາຢາຕ້ານເຊື້ອ ຫຼືຢາຢອດຕາ (ເຊັ່ນ: Hostacycline) ເປັນເວລາ 3–4 ມື້.
  • ໃຊ້ສີດພົ່ນຕາເພື່ອຂ້າເຊື້ອ.
  • ການປ້ອງກັນ : ຄວບຄຸມແມງວັນ ແລະແມງໄມ້; ຂ້າເຊື້ອໃນຖັງ ແລະອຸປະກອນທີ່ປົນເປື້ອນ.

○ ສານພິດໄຊຢາໄນ

  • ສາເຫດ : ການບໍລິໂພກພືດທີ່ມີໄຊຢານີເຈນ (ເຊັ່ນ: ໃບຊາ, ຍອດອ່ອນ).
  • ອາການ : ມີນ້ຳລາຍເປັນໂຟມ, ຊັກ, ຫາຍໃຈຍາກ, ມີເລືອດອອກທາງດັງ ແລະ ປາກ ກ່ອນເສຍຊີວິດ.
  • ການປິ່ນປົວ :
  • ອ່ອນໆ : ອາດຈະຟື້ນຕົວຕາມທໍາມະຊາດ.
  • ຄວາມຮຸນແຮງ : ປະຕິບັດ 1–2 ມລຂອງ 20% sodium nitrite IV ແລະ 3–5 mL ຂອງ 20% sodium thiosulfate IV.
  • ການປ້ອງກັນ : ຫ້າມບໍ່ໃຫ້ແບ້ເຂົ້າຫຍ້າໃນພືດທີ່ມີສານໄຊຢາໄນ.
